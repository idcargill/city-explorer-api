# City Explorer API

**Author**: Ian Cargill\
**Version**: 1.0.0

## Overview

An api backend for the City-Explorer app. Processes api request for movie and weather information.

## Getting Started

npm run server\
Will acitvate your server for testing.

## Architecture

- express.js
- dotenv
